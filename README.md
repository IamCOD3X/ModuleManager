# Module Loader

### Requirements

- Android 5.1+
- Root (Magisk 23.0+)
- Modules in **/system/lib/modules/**

### Features
1. Dynamically display available modules
2. Insmod/Rmmod modules
3. Detect active modules
4. Detect if module is loaded\unloaded successfully
5. Load modules on boot
